# 識愛 · Practice of Loving

一個關係練習提問卡的「抽卡」網頁，共 28 張卡（中英對照、手繪植物插圖）。

## 使用方式

直接開啟 `index.html` 即可離線使用，或透過 GitHub Pages 上線。

- 點卡背或按「抽一張」翻牌揭曉隨機卡片
- 一輪 28 張不重複，抽完按「重新洗牌」開新一輪
- 支援手機 / 桌機 / 鍵盤（Enter、空白鍵）

## 結構

```
index.html      抽卡頁面（自包含，無外部相依）
cards/          28 張卡片圖（01.png ~ 28.png）
```

## 啟用 GitHub Pages

Repo → Settings → Pages → Source 選 `main` 分支、`/ (root)` 目錄，
稍候即可透過 `https://chia1998.github.io/practiceofloving/` 瀏覽。
